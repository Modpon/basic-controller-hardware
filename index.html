Pump Control

  • Outputs: 3
  • Working: identical, they do the same thing unless you custom code
  • Power: 6-12v
  • Control:
    • Run time
    • Off time
    • Power

Lights control

  • Power: 12v
  • Control: on- off in code

Sensors

  • DHT11 or DHT22 temperature and humidity, visible on the screen
  • LDR, a basic light intensity meter, to do something..

Other info

  • Power: needs 12v adapter with 5mm plug
  • Screen: 128x128px TFT Oled
  • Controller: Arduino Nano with usb mini to connect to a pc
  • Software: Arduino language, based on c/ c++

Look in the Basic Controller Code repository for the Arduino code.
